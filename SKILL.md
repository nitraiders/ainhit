# SKILL: AINHITプロジェクト (インダストリアル・プレミアムサイト)

## 獲得スキル・実装パターン

### 1. 商業レベルのインダストリアル・ダークテーマへの刷新
*   **デザイン刷新**: 既存の安価なテンプレート色（ライトブルー等）とグラスモディフィズムを完全廃棄し、重厚なミッドナイト・ディープネイビー（`#0a192f`〜`#0f172a`）とサファイア・ロイヤルブルー（`#005bac`）の2軸でプレミアムなインダストリアルダークモードを構築するスキル。
*   **ロゴスタンドアロン配置**: ロゴ画像周囲の不細工な黒い背景ボックス（黒座布団）を完全撤去し、白抜き透過ロゴをダーク背景に透過状態でダイレクトに配置することで、クリーンな透明感と高級感を極限まで引き出す美的レイアウトスキル。

### 2. 極上のUI/UXディテール制御
*   **ヘッダースクロール遷移**: スムーズなJavaScriptトランジションとTailwind/CSSを組み合わせ、初期状態のダーク透過（`bg-slate-900/40 backdrop-blur-md`）からスクロール時の重厚なディープネイビーソリッド（`bg-slate-950/95`）へと滑らかに変化させ、テキスト視認性を永続維持するインタラクション設計。
*   **インサイドシャドウ・フォーカス連動**: お問い合わせフォームの入力フォーカス時に、プレーンなホワイトからロイヤルブルーへのスムーズな境界線変化およびインサイドシャドウと外側サファイアシャドウ（`0 0 25px rgba(0, 91, 172, 0.15)`）を連動させる高品位なフォームUIデザイン。
*   **プラチナヘアライン・グリッド**: 会社情報ブロックにて、ダミーテキストを排除し、プラチナシルバーのヘアライン加工を想起させる背景グラデーション（`from-slate-900 to-slate-950`）と微細グリッドパターンを重ね合わせたエンブレム装飾枠の構築。

### 3. Gitベースの堅牢なクラウドインフラ統合（絶対ルール）
*   **Git連携運用の遵守**: 
    *   Wrangler CLIを用いた一時的・場当たり的な直接コマンドデプロイ（`wrangler pages deploy`）だけでプロジェクトを完了扱いにせず、インフラの品質担保として「**GitHub repository → Cloudflare Pages Git連携**」を必須のデプロイ自動化基準とする。
    *   この設計により、開発者が `main` ブランチへ `git push` するだけで、安全かつ安定したビルドプロセスがCloudflare側で自動トリガーされ、差分管理と永続運用がシームレスに行われる体制を標準スキルとして定義。
